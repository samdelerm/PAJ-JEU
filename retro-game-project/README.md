# Retro Game Project

Welcome to the Retro Game Project! This project features a collection of four fun and engaging mini-games, each with its own unique theme and gameplay mechanics. Below is a brief overview of each level and instructions on how to run the game.

## Levels Overview

### Level 1: Beach Volleyball Game
- **Description**: Play a classic beach volleyball game where you can control a player to serve, spike, and block against an opponent.
- **Objective**: Score points by winning sets against the AI opponent.

### Level 2: Camping-Themed Pac-Man Style Game
- **Description**: Navigate through a maze filled with camping-themed obstacles while collecting pellets and avoiding ghosts.
- **Objective**: Collect all the pellets to advance to the next level while avoiding being caught by the ghosts.

### Level 3: Archery Game
- **Description**: Test your aiming skills in this archery game where you shoot arrows at moving targets.
- **Objective**: Score as many points as possible by hitting the targets within a time limit.

### Level 4: Kart Racing Level
- **Description**: Race against opponents on a kart racing track filled with power-ups and obstacles.
- **Objective**: Complete laps and finish in the top position to win the race.

## Getting Started

To run the game, follow these steps:

1. Clone the repository or download the project files.
2. Open the `index.html` file in your web browser.
3. Use the arrow keys and other specified controls to navigate through the games.
